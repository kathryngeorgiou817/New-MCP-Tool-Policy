# CDSSpinner

Import from `@ciscodesignsystems/cds-react-spinner`.

For full props and usage, use the Magnetic MCP tool `mcp__magnetic-dev__ask_react_components` to query this component's documentation.

Loading spinner indicator.

```tsx
<CDSSpinner size="md" />
```

- `size` — `"sm"` | `"md"` | `"lg"`.
- For a centered loading state, wrap in `<CDSFlex align="center" justify="center">`.
